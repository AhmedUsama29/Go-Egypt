## 🚀 Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local web server for development

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AhmedUsama29/Go-Egypt.git
   cd Go-Egypt
   ```

2. **Open the project:**
   - Simply open `about-us.html` in your web browser
   - Or use a local server for development:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **View the website:**
   - Direct file: Open `about-us.html` in browser
   - Local server: Visit `http://localhost:8000`

## 🎨 Customization

### Colors
The main color scheme can be customized in `style.css`:
```css
:root {
  --primary-color: #ff8c00;    /* Orange/Gold */
  --secondary-color: #2c3e50;  /* Dark Blue */
  --accent-color: #d4af37;     /* Gold */
  --text-color: #333;          /* Dark Gray */
}
```

### Content
- Update team member information in `about-us.html`
- Modify company story and mission in the Our Story section
- Customize navigation links and footer content

### Images
- Replace placeholder team member images with actual photos
- Add company logo and story images
- Update image paths in HTML file
